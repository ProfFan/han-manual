漢字標準格式

印刷品般的漢字排版框架

  • GitHub Project

簡介

「漢字標準格式」是一個集合了「語意樣式標準化」「文字設計」「高級排版功能」等三大概念的網頁排版框架，使用Sass及JavaScript構架而成。其專為漢字網頁提供的美觀而標準化的環境，不僅符合傳統閱讀習慣、更為螢幕閱讀提供了既成標準，得以完整解決現今漢字網頁設計的排版需求。

「漢字標準格式」支援繁體中文、簡體中文及日文等三個採用漢字的語言文字。

用途與特色

完全相容HTML5，簡潔而不冗餘

「漢字標準格式」嚴格遵守HTML5各種元素的語意，不使用冗餘繁雜的展示性類別（presentational classes），將元素的預設樣式重新定製、標準化，更適合漢字的書寫排印習慣。

不同文字、不同字體

配合精細而完整的字體回退機制，「漢字標準格式」得以依據不同的語言、文字，以字元為單位，套用其最合適的字體。從此標點符號不再混亂，漢字、西文、數字各得其所。

專為「舒適閱讀」打造

諸多高級排版功能——如「漢字－西文混排間隙」「標點擠壓」「行尾點號懸掛」「章節邊界調整」等等——皆為舒適閱讀打造，體驗如若印刷品般的螢幕閱讀。

以設計流線、而非像素呈現

「漢字標準格式」重視語意、設計流線和彈性，所有樣式的呈現皆避免了以像素為單位的靜態設計。
